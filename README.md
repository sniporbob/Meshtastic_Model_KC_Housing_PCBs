# Meshtastic_Model_KC_Housing_PCBs
PCB files for the Model KC meshtastic housing. There is an internal board that uses 2 target discs and gets epoxied into the base of the housing, and there's a charger board that holds a pair of pogo pins and allows soldering wires from a USB cable.

0.118" x 0.065" Target discs: part number 1618-4-57-15-00-00-03-0 from Mill-Max

Pogo Pins: P75-E2 from Amazon - https://www.amazon.com/dp/B099F1DRYJ - 16.54mm total length, 2.54mm compression, 1.02mm shaft diameter

Files in this repository include KiCad 8 project files and pre-generated gerber files. I milled these boards myself on a 3018 CNC mill so there are also panel layout KiCad files which are probably not useful to anyone.
